我的第一篇博客

#这是关于我建立博客遇到的问题 在建立过程中，我尝试了三次。 ##第一次重建，是在创建本地hugo的时候，由于参考了多数文件，导致我步骤重复，反复报错，烦不胜烦。 ##第二次是想改博客主题，但没有找到方法。 ##第三次也遇到2个问题， 1.由于github因安全因素更新，使得用户名-邮箱不能上传，改用密钥。 所以需要在github建立SHH密钥，我建立好密钥，却反馈我连接不上，经老师提示，是我网络连接出了问题，我用的steam++，在改用fastgithub后，成功连接。 2.远程与本地的分支冲突（我本地是master，github托管网站却默认为main分支？），在请教老师和学生导师后，我使用git push origin main –force之后再用 git push origin main将本地文件上传到github的main分支进行托管，使得博客网站显示出本地的内容。
